# Roadmap

This roadmap is directional and may evolve based on stability, user feedback, and maintainer priorities.

## Now
- Canvas reliability and workflow ergonomics.
- Node inspector clarity and integration-specific UX.
- Run diagnostics and retry/resume usability.
- Packaging and release consistency (`.deb`, AppImage, Flatpak, portable).

## Next
- Execution engine hardening (timeouts, retries, replay from failed node).
- Integration credential profiles and stronger required-field guidance.
- Expanded starter templates and outcome-driven packs.
- Better operational dashboards and run timeline visibility.

## Later
- Plugin SDK and safer connector extension model.
- Team/workspace features and operational governance controls.
- Web edition architecture for cross-platform self-hosting.
- Hosted/managed deployment options.

## Proposal Guidance
- Please open an issue before major feature work.
- Link proposals to user value, risks, and migration impact.
